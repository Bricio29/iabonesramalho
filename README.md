# Bot Bonés Ramalho - IA WhatsApp

Bot de atendimento automatizado via WhatsApp com qualificação inteligente de leads e envio de imagens de produtos.

## 🚀 Instalação

```bash
npm install
npm start
```

## ⚙️ Configuração

Crie o arquivo `.env`:
```env
OPENAI_API_KEY=sua_chave_aqui
EQUIPE_NUMERO=5584999999999@c.us
```

## 📸 Imagens

Adicione as imagens dos produtos em:
- `assets/modelos/` - 15 modelos principais (BRAM02.jpeg, BRTR01.jpeg, etc)
- `assets/extras/` - Fotos adicionais
- `assets/logomarca/` - Exemplos de aplicação
- `assets/reguladores/` - Opções de regulador

Ver `assets/README.md` para detalhes.

## 🤖 Funcionalidades

- Qualificação inteligente de leads
- Recomendação personalizada de modelos
- Envio seletivo de imagens (2-3 modelos recomendados)
- Respostas sobre prazos, frete e pagamento
- Transbordo automático para humanos (pedidos 100+ unidades)
- Suporte a áudio (transcrição via Whisper)
